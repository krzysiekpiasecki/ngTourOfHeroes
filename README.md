# ngTourOfHeroes


Angular 2 app to help a staffing agency manage its stable of heroes.

### Quick start

```bash
$ git clone https://github.com/krzysiekpiasecki/ngTourOfHeroes.git
$ cd ngTourOfHeroes
$ npm install
$ ng serve

Go to: http://localhost:4200
```
